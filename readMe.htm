Welcome to the JS GUI 2.0, this page was created completely with this editor, in fact, if you wish you may add ?edit to the end of the url and edit this
page!
To add this functionality to any preexisting page, you need to add the following lines betweed the head tags: 

<script type="text/javascript" src="chainLoad.js"></script>
<script type="text/javascript">var idInc=1;var inEdit=false;checkIfInEdit();</script>


This will allow you to edit the page by adding a ?edit to the end of the url as long as the appropriate files
are in place!

Please note, objects in a premade page are most likely set as inline positioning, meaning they can't be
moved unless they are first set to absolute(which will remove them from the continuity). You can do this
by selecting the item, going to "Edit" and then "Properties" then checking the "Absolute" box.

To save a page simply go to File, and select "Finish&Save". This will output the code. You may copy this
code and save it to a new htm file using notepad and upload as needed.

You may also use the full version which allows direct uploads to the server with proper authentication.
